# Retail ETL Pipeline

Este proyecto implementa un pipeline ETL en **PySpark** usando el dataset de Online Retail.

## Objetivo
Construir un flujo ETL que:
1. **Extract:** lee datos de ventas desde un archivo CSV.
2. **Transform:** limpia registros inválidos y calcula métricas de negocio.
3. **Load:** guarda los resultados en formato Parquet, listos para análisis.

## Tecnologías
- Python
- PySpark
- SQL básico
- Formatos: CSV → Parquet

## Resultados
- **Ventas por cliente:** gasto total y número de facturas.
- **Ventas por producto:** ingresos y cantidad vendida.
- **Ventas por país:** ingresos totales y número de clientes.
